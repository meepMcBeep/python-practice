Python Practice

This repository contains structured Python exercises designed to rebuild
and strengthen core programming skills.

The focus is on: - Clean Python syntax - Progressive complexity -
Problem-solving fundamentals - Consistent daily practice

------------------------------------------------------------------------

📚 Structure

Exercises are organized into sets, each focusing on a specific concept:

python/ ├── set1/ – Basic syntax, variables, printing, input ├── set2/ –
Conditionals and control flow ├── set3/ – Loops and iteration ├── set4/
– Functions ├── set5/ – Data structures ├── set6/ – (upcoming)

Each file represents a single assignment and includes: - The problem
statement (as a comment) - The Python solution

------------------------------------------------------------------------

🎯 Goals

-   Rebuild Python fluency from fundamentals
-   Develop clean and readable coding habits
-   Strengthen debugging and problem-solving skills
-   Practice real-world programming patterns used in scripting,
    automation, and data workflows

------------------------------------------------------------------------

🧠 Learning Approach

-   Small, focused exercises (10–15 minutes)
-   Emphasis on recall over recognition
-   Minimal hints to encourage independent problem solving
-   Iterative improvement through review and refinement

------------------------------------------------------------------------

🛠️ Environment

Exercises are developed and tested using: - Local terminal and text
editor / IDE - Git and GitHub for version control - Future expansion to
Python-based data and cloud workflows

------------------------------------------------------------------------

✍️ Python Style Guidelines

-   Use clear and descriptive variable names
-   Prefer readable, simple solutions over clever ones
-   Format code consistently
-   Add comments when they improve clarity
-   Pay attention to errors, stack traces, and debugging workflow
